    Aims-

  • The mission of this course is to teach students how to build computer systems that can be trusted in situations where the system�s response to external events must be both timely and accurate. An important class of such systems is the systems involved to control physical processes. Students will be taught how conventional (analog) control systems can be implemented by software running on digital computers. Real-time operating systems will be discussed and students will learn how to write drivers for the Linux kernel using RTAI (Real Time Application Interface). They will also be introduced to the task model and some real-time scheduling algorithms. Specification and design of real-time systems will also be discussed.
